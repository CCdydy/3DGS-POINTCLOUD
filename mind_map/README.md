# Mind Map

这个目录用于集中存放项目的调研材料、阶段性总结和研究定位草稿。

## 文档索引

### [当前进展总览.md](当前进展总览.md)

- 类型：中文总览页
- 语言：中文
- 作用：快速说明当前项目结构、文档分工、已整理内容和推荐阅读顺序
- 适合场景：重新接手项目、快速回顾当前状态、给别人解释仓库结构

### [阶段总结_中文.md](阶段总结_中文.md)

- 类型：中文阶段总结
- 语言：中文
- 作用：概括当前项目处于什么阶段、已完成什么、风险在哪里、接下来该做什么
- 适合场景：阶段汇报、口头说明、快速回顾项目状态

### [结果记录模板.md](结果记录模板.md)

- 类型：记录模板
- 语言：中文
- 作用：统一登记场景、压缩参数、训练配置和评估结果
- 适合场景：补实验记录、做结果表、整理 sensitivity curve 原始数据

### [compass_artifact_学术调查.md](compass_artifact_学术调查.md)

- 类型：系统调研 + 可复现实验路线
- 语言：英文正文，中文文件名
- 作用：梳理 LiDAR 压缩、自动驾驶 3DGS、3DGS 压缩三条研究线，整理 baseline、数据集、依赖和实验切入点
- 适合场景：开题、文献梳理、确定 baseline、制定复现顺序

### [复现后工作.md](复现后工作.md)

- 类型：研究定位与问题定义草稿
- 语言：英文正文，中文文件名
- 作用：从论文切入角度概括研究 gap、最接近工作、风险评估和论文入口
- 适合场景：写 proposal、写 paper intro、整理 research pitch
